Anthropic Daily · 2026-04-21

Anthropic 全系 SDK 统一更新支持 Claude Opus 4.7，Claude Haiku 3 正式退役；Claude Code Action 发布 v1.0 正式版大幅简化配置；Dario Amodei 就 AI 可解释性与技术发展发表多篇重要文章。

平台与 API 更新

**Claude Haiku 3 正式退役** — 2026年4月20日起，Claude Haiku 3（claude-3-haiku-20240307）已停止服务，所有调用该模型的请求将返回错误。Anthropic 建议用户升级至 **Claude Haiku 4.5**。
Claude Platform Release Notes

开发者工具更新

**Claude Agent SDK (Python)** 发布多个版本： - **v0.1.64**：完整 SessionStore 适配器支持，包含 `append`、`load`、`list_sessions` 等5个协议方法，新增 `--session-mirror` 转录镜像功能，以及9个异步存储辅助函数。 - **v0.1.63**：更新内置 Claude CLI 至 v2.1.114。 - **v0.1.62**：新增 `skills` 参数，可通过 `"all"`、命名技能列表或 `[]` 控制技能启用。 **Claude Code Action** 正式发布 **v1.0 正式版**，带来配置大幅简化：取消手动 mode 配置，改为**自动检测 interactive/automation 模式**。此外更新至 v1.0.102，升级 oven-sh/setup-bun 至 v2.2.0（支持 Node.js 24）。 **Anthropic CLI** 发布 v1.2.1 和 v1.2.0，新增参数别名（`x-stainless-cli-data-alias`）功能。

系统提示词更新

Anthropic 持续优化各 Claude 模型的 System Prompt，主要更新包括： - **Claude Opus 4.7**（4月16日）：系统提示词更新 - **Claude Sonnet 4.6**（2月17日）：系统提示词更新 - **Claude Haiku 4.5**（1月18日、11月19日、10月15日）：多轮更新 - **Claude Sonnet 4.5**（1月18日、11月19日、9月29日）：多轮更新 - **Claude Opus 4.6**（2月5日）：系统提示词更新 - **Claude Opus 4.1/4/3**、**Sonnet 4/3.7/3.5**、**Haiku 3.5/3**：历史版本更新记录 System Prompt 用于在对话开始时向 Claude 提供实时信息（如当前日期），并引导特定行为（如始终以 Markdown 格式提供代码片段）。

高管观点与行业评论

可解释性研究进展

Anthropic 持续深耕 AI 可解释性（Interpretability）领域，公开多项研究成果： - **Activation Oracles** — 训练语言模型用自然语言回答关于自身激活的问题 - **Automated Auditing** — 利用 Agent 执行自动化对齐审计，结合可解释性工具 - **PySvelte** — 连接 Python 与 Web 交互图表的基础设施，支持可解释性研究可视化 此外还收录了相关经典文献：Distill 的 **Circuits Thread**（Transformer 电路解释奠基工作）、**Activation Atlases**（激活图谱）等。
