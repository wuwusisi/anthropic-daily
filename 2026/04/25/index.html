Anthropic Daily · 2026-04-25

Anthropic 今日更新聚焦开发者体验：Claude Code 增强配置持久化和多平台 PR 支持，Agent SDK 修复 Trio 兼容性并改进长会话稳定性，Rust 生态工具链持续完善。

开发者工具链更新

**Claude Code v2.1.119** 带来多项实用改进： - `/config` 设置（主题、编辑器模式、verbose 等）现在持久化到 `~/.claude/settings.json`，支持项目级、本地级、策略级覆盖优先级 - 新增 `prUrlTemplate` 设置，支持自定义 PR badge 指向的代码审查 URL（可对接内部工具） - 新增 `CLAUDE_CODE_HIDE_CWD` 环境变量，可在启动 Logo 中隐藏工作目录 - `--from-pr` 参数现在支持 GitLab merge-request 和 Bitbucket pull-request URL **这意味着什么**：对于在 CI/CD 流程中集成 Claude Code 的团队，这些改进提升了配置管理的灵活性和工具的适配性。
claude-code v2.1.119 Release

Agent SDK 更新

**claude-agent-sdk-typescript v0.2.119** 改进长会话稳定性： - MCP 服务器重连机制增强，断连后可自动恢复 - `SessionStore.append()` 失败时自动重试 3 次，降低数据丢失风险 **claude-agent-sdk-typescript v0.2.118** 新增企业级功能： - 新增 `Options.managedSettings` 选项，支持嵌入方传递策略层级的配置 **claude-agent-sdk-python v0.1.67** 修复重要兼容性问题： - 恢复了对 Trio 异步框架的兼容支持，解决了 v0.1.51 引入的回归问题 **这意味着什么**：这些修复和增强主要面向构建 AI Agent 应用的开发者，特别是需要长时运行、复杂异步处理或企业级策略管控的场景。
claude-agent-sdk-typescript v0.2.119 claude-agent-sdk-typescript v0.2.118 claude-agent-sdk-python v0.1.67

基础设施与协议库

Anthropic 今日开源了 **tokio**（Rust 异步运行时仓库）和 **connect-rust**（ConnectRPC 协议 Rust 实现），后者迎来 v0.3.3 版本更新： - 新增 streaming-tour 和 middleware 使用示例 - 上线用户指南和 adoption 帮助文档 - 完善 CI 流程，切换到 arduino/setup-protoc action **这意味着什么**：Anthropic 持续投入 Rust 生态建设，为需要高性能、低延迟 AI 应用的开发者提供更完善的底层基础设施。ConnectRPC 作为与 gRPC 兼容的协议选择，其文档完善降低了接入门槛。

模型 System Prompt 更新

Claude Opus 4.5、Claude Haiku 4.5、Claude Sonnet 4.5 均于 2026 年 1 月 18 日更新了 System Prompt。这是新年以来的首次系统性更新，具体变更内容需参考官方 Release Notes。 **这意味着什么**：System Prompt 的调整可能影响模型在特定任务场景下的行为表现，建议开发者关注后续官方说明。
