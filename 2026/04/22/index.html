Anthropic Daily · 2026-04-22

Anthropic 全系 SDK 同步新增 Token Budgets、User Profiles 及 claude-opus-4-7 模型支持，同时标记 Sonnet/Opus 4 弃用，六语言官方 SDK 生态正式落成，补全 .NET 和 Ruby 支持。

API 功能更新

Anthropic 主流 SDK（Python、TypeScript、Java、Go、Ruby、C#）同步新增三项重要 API 能力： - **claude-opus-4-7**：全新模型标识符上线 - **Token Budgets**：支持 Token 消耗预算管理，开发者可设置和监控 API 调用额度 - **User Profiles**：用户配置文件功能，支持个性化配置管理 **这意味着什么**：Token Budgets 功能对需要精细控制成本的企业用户是重要利好，可实现按项目/团队分配额度；User Profiles 为多租户场景提供了原生支持，降低了企业级应用开发门槛。
Python SDK v0.96.0 TypeScript SDK v0.90.0 C# SDK Anthropic-v12.16.0

模型生命周期变更

各 SDK 同步标记 **Sonnet** 和 **Opus 4** 为 deprecated（旧版标识符）。 **这意味着什么**：使用旧版模型标识符的开发者需尽快迁移至新版本。建议在产品文档中明确标注迁移时间线，避免影响现有用户。Claude Sonnet 和 Opus 4 的正式退役可能临近，有相关依赖的产品应提前规划。
Python SDK v0.95.0 TypeScript SDK v0.89.0 Go SDK v1.36.0

开发者生态：多语言 SDK 正式发布

Anthropic 宣布发布六种编程语言的官方 SDK，覆盖主流开发场景： | 语言 | 仓库 | 定位 | |------|------|------| | Python | anthropic-sdk-python | 主力语言 | | TypeScript | anthropic-sdk-typescript | 前端/Node.js | | Java | anthropic-sdk-java | 企业级 | | Go | anthropic-sdk-go | 云原生 | | Ruby | anthropic-sdk-ruby | Web 全栈 | | C# | anthropic-sdk-csharp | .NET 生态 | 此外，C# SDK 新增 **beta tool runner** 功能；Java SDK 新增 **beta advisor tool**，进一步丰富 Agent 工具生态。 **这意味着什么**：官方多语言 SDK 的全面发布降低了集成门槛，企业可根据技术栈自由选择。.NET 和 Ruby 生态的加入补全了之前的功能缺口，有助于 Anthropic 拓展企业客户群体。

云平台支持扩展

- **Vertex AI EU Region**：TypeScript SDK 和 Java SDK 新增 Vertex 欧洲区域支持 - **Bedrock Mantle Client**：多语言 SDK 统一使用 auth header 方式接入 Bedrock Mantle **这意味着什么**：EU 区域支持满足了数据本地化合规需求，对于欧洲市场的企业客户是重要能力；Bedrock 接入方式的统一降低了多云部署的复杂度。
