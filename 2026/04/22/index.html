开发者工具

Anthropic 今日发布/更新了多个开发者工具仓库，重点包括： **新版 Claude Code Action (v1.0) 正式发布**：该 GitHub Action 引入自动模式检测功能，无需手动配置 mode 参数，智能识别交互式或自动化运行环境。新增工作流语法验证、改进的错误处理和安全加固，支持在 CI/CD 流水线中无缝集成 AI 辅助开发。 **Claude Code v2.1.116-v2.1.117 主要更新**： - 新增 forked subagent 支持（通过 `CLAUDE_CODE_FORK_SUBAGENT=1` 启用） - `/resume` 命令在大型会话场景下提速达 67% - MCP 服务器启动优化，多 stdio 服务器配置场景更高效 - Agent frontmatter `mcpServers` 现已支持主线程 agent 会话 - 模型选择持久化改进，项目 pin 设置更加灵活 **Claude Agent SDK 更新**： - Python SDK v0.1.64 新增完整 SessionStore 支持，包含 5 个核心方法（append、load、list_sessions、delete、list_subkeys）及内存实现，新增 9 个异步存储辅助函数，支持会话镜像和恢复功能 - TypeScript SDK 同步更新至 v0.2.117/116/114，与 Claude Code 保持版本对齐 **新仓库发布**：Claude Cookbooks（使用指南）、Claude Plugins Official（官方插件目录）、Knowledge Work Plugins（知识工作者插件集）等开发者资源库陆续上线。

AI 安全与对齐研究

**Activation Oracles**：Anthropic 发布新研究项目，探索训练语言模型用自然语言回答关于自身激活（activations）的问题。这一方向有助于提升模型的可解释性（Interpretability），让研究者更好地理解模型内部运作机制。 **Automated Auditing**：探讨使用 AI agents 执行自动化对齐审计（Alignment Audits）的方法，涵盖可解释性工具在审计流程中的应用，为 AI 安全评估提供新思路。 **PySvelte**：Anthropic 开源基础设施工具，用于桥接 Python 与基于 Web 的交互式图表，辅助可解释性研究中的可视化工作。

行业观点与媒体动态

**Dario Amodei 发表多篇深度文章**：Anthropic CEO Dario Amodei 近期在个人博客发布系列文章，涵盖： - 《The Adolescence of Technology》——探讨技术发展的青春期阶段 - 《Machines of Loving Grace》——论 AI 与人类价值观对齐 - 《The Urgency of Interpretability》——强调可解释性研究的紧迫性 - 《On DeepSeek and Export Controls》——分析 DeepSeek 事件与出口管制政策 **政策与监管观点**：Dario Amodei 在《纽约时报》发表署名文章《AI Needs Basic Transparency Rules》，呼吁建立 AI 基础透明性规则；在《华尔街日报》发表《Trump Can Keep America's AI Advantage》，阐述美国如何保持 AI 竞争优势。 **媒体采访**：Dario Amodei 近期接受了 TIME 专访及多个播客节目采访，包括 Lex Fridman Podcast、The Ezra Klein Show、Dwarkesh Podcast 等，讨论 AI 发展、安全与治理等议题。
